# HTML `<video>` ტეგი

| ატრიბუტი | აღწერა | მნიშვნელობა |
|-----------|-------------|----------------|
| src | ვიდეოს მისამართი | URL |
| autoplay | ვიდეოს რთავს ავტომატურად | boolean |
| controls | აჩვენებს ვიდეოზე სხვადასხვა საკონტროლო საშუელებას | boolean |
| loop | ამეორებს ვიდეოს დასრულების შემდეგ | boolean |
| muted | ნაგულისხმევად ვიდეოს უხმოდ ჩატვირთვა  | boolean |
| poster | ეთითება სურათი რომელიც გამოჩნდება ვიდეოს ჩართვამდე | URL |
| width | ვიდეოს სიგანე | integer |
| height | ვიდეოს სიმაღლე | integer |
| preload | ვიდეოს წინასწარი ჩატვირთვა | none, metadata, auto |

# ვიდეოს, ფოტოს და აუდიო ფაილების გადმოსაწერი ლინკები:
# - [pixabay.com](https://pixabay.com/videos/)
