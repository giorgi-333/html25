
# HTML ფორმები - შესავალი

| ტეგი | აღწერა | ატრიბუტები |
|---|---|---|
| `<form>` | ფორმის შექმნა | action, method, name, target |
| `<input>` | სხვადასხვა ტიპის ველების შექმნა | type, name, value
| `<textarea>` | ტექსტის მრავალსვეტიანი ველი | name, rows, cols |
| `<button>` | ღილაკის შექმნა | type, name, value |

### `form-ის ატრიბუტები:`
- `action:` განსაზღვრავს URL-ს, სადაც ფორმის მონაცემები გაიგზავნება.
- `method:` განსაზღვრავს HTTP მეთოდს (GET ან POST) ფორმის მონაცემების გადასაცემად.
- `name:` ფორმის სახელი, რომელიც მნიშვნელოვანია ბექისთვის.
- `target:` განსაზღვრავს, სად გაიხსნება ფორმის შედეგი (მაგალითად, _blank ახალი ფანჯარა).
### `textarea-ს და input-ის საერთო ატრიბუტები:`
- `name:` ტექსტის ველის სახელი.
- `value:` ველის საწყისი მნიშვნელობა.
- `placeholder:` ველის შიგნით ნაჩვენები ტექსტი, მნიშვნელობების ჩაწერამდე.
- `disabled:` ველში ჩაწერის შესაძლებლობის გაუქმება.
### `input-ის ატრიბუტები:`
- `type:` განსაზღვრავს ველის ტიპს, როგორიცაა:
    - `text:` ერთსვეტიანი ტექსტის ველი.
    - `password:` პაროლის ველი, სადაც ტექსტი დამალულია.
    - `email:` ელფოსტის ველი, რომელიც ვალიდაციას უკეთებს ელფოსტის ფორმატს.
    - `submit:` ღილაკი ფორმის გაგზავნისთვის.
    - `reset:` ღილაკი ფორმის ველების დასარესეტებლად.
    - `checkbox:` ჩეკბოქსი, რომელიც საშუალებას აძლევს მომხმარებელს აირჩიოს რამდენიმე ვარიანტი.

    <!-- fieldset, legend  -->
textarea-ის ატრიბუტები:
### `textarea-ის ატრიბუტები:`
- `name:` ტექსტის ველის სახელი.
- `rows:` ტექსტის ველის სტრიქონების რაოდენობა.
- `cols:` ტექსტის ველის სვეტების რაოდენობა. 
### `button-ის ტიპები:`
- `name:` ღილაკის სახელი.  
- `type:` განსაზღვრავს ღილაკის ტიპს (submit, reset, button).
- `value:` ღილაკის მნიშვნელობა.
